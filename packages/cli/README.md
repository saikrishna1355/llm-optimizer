# @llm-optimize/cli

`@llm-optimize/cli` provides command-line tools for prompt analysis and linting.

## Install

```bash
npm install @llm-optimize/cli
```

## Use

```bash
npx llm-opt analyze prompt.md
npx llm-opt lint prompts/
npx llm-opt benchmark
```

## Why use it

Use this package to inspect prompts before sending them to an LLM.
